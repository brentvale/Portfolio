##### Setup
Clone the repo and run ```npm install```.  

##### Running Tests
Jest test suite can be run with ```npm test``` or ```yarn test```

##### Lint
Code can be linted with ```yarn lint``` and get rid of those pesky little lint errors with ```yarn lint --fix```
